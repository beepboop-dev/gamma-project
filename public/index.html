API Reference

Base URL: https://gamma.abapture.ai/v1 · Auth: x-api-key header or api_key query param

GET/validate/email?email={email}
Validate an email address format and extract domain
curl "https://gamma.abapture.ai/v1/validate/email?email=hello@example.com" -H "x-api-key: YOUR_KEY" → { "email": "hello@example.com", "valid": true, "domain": "example.com" }
GET/meta/url?url={url}
Extract title, description, OG image, and favicon from any URL
curl "https://gamma.abapture.ai/v1/meta/url?url=https://github.com" -H "x-api-key: YOUR_KEY" → { "title": "GitHub", "description": "...", "image": "...", "favicon": "..." }
GET/generate/qr?text={text}&format=svg|json
Generate QR codes as SVG or base64 data URL
curl "https://gamma.abapture.ai/v1/generate/qr?text=https://example.com&format=svg" -H "x-api-key: YOUR_KEY" → (SVG image)
GET/generate/hash?text={text}&algorithm=sha256
Hash text with SHA-256, MD5, SHA-512, etc.
curl "https://gamma.abapture.ai/v1/generate/hash?text=hello&algorithm=sha256" -H "x-api-key: YOUR_KEY" → { "hash": "2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824" }
GET/generate/uuid?count={1-100}
Generate one or more UUIDs
curl "https://gamma.abapture.ai/v1/generate/uuid?count=3" -H "x-api-key: YOUR_KEY" → { "count": 3, "uuids": ["...", "...", "..."] }
POST/analyze/text
Word count, reading time, sentence count, avg word length
curl -X POST "https://gamma.abapture.ai/v1/analyze/text" \ -H "x-api-key: YOUR_KEY" -H "Content-Type: application/json" \ -d '{"text": "Hello world. This is a test."}' → { "characters": 28, "words": 6, "sentences": 2, "reading_time_seconds": 2 }
GET/convert/timestamp?ts={unix}&date={iso}
Convert between Unix timestamps and ISO dates (no params = now)
curl "https://gamma.abapture.ai/v1/convert/timestamp?ts=1700000000" -H "x-api-key: YOUR_KEY" → { "unix": 1700000000, "iso": "2023-11-14T22:13:20.000Z" }
GET/lookup/ip
Get the requester's IP address
curl "https://gamma.abapture.ai/v1/lookup/ip" -H "x-api-key: YOUR_KEY" → { "ip": "203.0.113.42" }
